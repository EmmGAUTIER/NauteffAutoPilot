◆ MessageMotorType_t

enum MessageMotorType_t

Callback of timer and sends requests to taskMotor.

This callback function sends MSG_MOT_MOVING_CONTROL periodicaly to taskMotor when motor is running. When taskMotor receive this message it measures current compute an estimated position and checks current.
