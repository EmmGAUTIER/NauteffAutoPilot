# NauteffAutoPilot - Documentation

## Objectifs

### Pour les utilisateurs

Connaître les objectifs du Nauteff et ses caractéristiques.

### Pour les programmeurs

Comprendre l'architecture du logiciel,
connaître les paradigmes de développement,
retrouver les fonctions et leurs sources et
compiler le source.

## État d'avancement

Après de nombreux essais et recherches la définition  du Nauteff,
son architecture et son code se précisent. Sa rédaction est en cours.
Le premier essai en mer a permis de confirmer sa faisabilité.
Le document Nauteff-Spécification.pdf contient une description.
C'est le document recommandé pour une appréhension du projet.
